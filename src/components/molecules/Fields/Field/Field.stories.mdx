# Field

This is the `Input` component with xstyled super powers and another elements inside doing some magics.

## Props

- `label`: string

  The label of input showed in the top-left corner of the field container.

- `error`?: string | undefined | null

  The error message showed in the bottom-left corner of the field container.

- `browserAutoComplete`?: string | undefined

  The react input prop `autoComplete` was renamed for it.

- `prefixElement`?: ReactNode

  An element to be shown before input inside the field container.

- `sufixElement`?: ReactNode

  An element to be shown after input inside the field container.

- `fileTypeOptions`?: {
  accept?: string,
  inputMessage: string
  }

- `type`?: 'date'
  | 'datetime-local'
  | 'month'
  | 'number'
  | 'password'
  | 'search'
  | 'text'
  | 'time'
  | 'week'
  | 'email'
  | 'url'
  | 'file'
