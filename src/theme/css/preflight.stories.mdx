# Preflight

This is a simple CSS file used for resets and consistency in the style of the site.

## Usage

In your `_app.tsx` (next.js) or root component:

```ts
import '@nexpy/design-system/dist/preflight.css'
```

or

```ts
import '@nexpy/design-system/dist/preflight.min.css'
```
